# Inbox.Video

## As of Nov 2020 this repo is currently being deprecated.

----


![Screenshot](./app/images/screenshot.png)

[Inbox.Video](http://inbox.video/) is a new way to watch Youtube videos.

## Build

Install the node modules:

    $ npm install

Develop with:

    $ npm run dev

Build with:

    $ npm build
